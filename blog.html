1. Difference between map(), forEach(), filleter(), find() method?

Ans: map(), forEach(), filleter() and find() are method of array data type. They are work form different angle.

  1. map() method work with each element of array and return a new array.
  2. forEach() method work with each element of array but don't return anything.
  3. filleter() method work with a each element with conditions, it return a new array of elements which are follow the conditions.
  4. find() method work with array with conditions, it return first element (not as array) of that array which are follow the conditions.
2. Difference between local storage, session storage, cookies?

Ans: Local storage, session storage are for client side and Cookies is for both client and server side.

  1. Local Storage store data for a longer period of time in client side. Even after the browser or computer is closed the data is available, and the data remains available until it is manually cleared by the user or the application.
  2. Session Storage store data only for the duration of the user's session on a website in client side, and the data is cleared when the session ends or the browser is closed.
  3. Cookies store a small amounts of data that need to be accessed by both the client and server. And the data remains available until it is manually cleared by the user or the application.
3. What is event Loop ?

Ans: Event Loop is a JavaScript code execution system. This system is execute js code in an asynchronous and non-blocking way. Its a lopping algorithm. There is non-stop code execution model. If any function is waiting code it go to task queue and code execution are run by its own. when that function waiting end its come to call stack and execute.
